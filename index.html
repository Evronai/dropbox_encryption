Powerful Features

AES-256 Encryption

Military-grade encryption using Fernet (symmetric encryption) with PBKDF2 key derivation.

Dropbox Integration

Automatic upload to Dropbox with shareable links. Chunked upload for large files.

Telegram Bot

Interactive bot with inline buttons. Encrypt/decrypt files directly from Telegram.

Dual Encryption Modes

Master key for convenience or password-based with unique salt per file.

Auto Link Copying

Shareable links automatically copied to your clipboard after upload.

Beautiful CLI

Colorful, intuitive command-line interface with clear menus and feedback.
